# GPIO 输入与按键消抖学习笔记

## GPIO 输入模式

| 模式 | 特点 | 用途 |
|---|---|---|
| 浮空输入 | 无默认电平，容易受噪声影响 | 外部已提供确定电平 |
| 上拉输入 | 默认保持高电平 | 按键按下接地 |
| 下拉输入 | 默认保持低电平 | 按键按下接 3.3 V |
| 模拟输入 | 关闭数字输入路径 | ADC 输入 |

## 为什么输入不能悬空

悬空 GPIO 缺乏确定电平，可能受噪声影响而随机跳变，因此通常需要内部或外部上拉、下拉电阻。

## 高有效与低有效

高有效：

```text
未按下：0
按下：1
```

低有效：

```text
未按下：1
按下：0
```

本实验使用内部上拉，按下接地，因此属于低有效。

## 非阻塞消抖

1. 原始电平变化时记录时间。
2. 电平稳定不足 20 ms 时不更新状态。
3. 稳定达到 20 ms 后更新状态。
4. 仅在状态边沿产生一次事件。

## 状态与事件

当前状态：

```c
gpio_button_is_pressed(&button);
```

按下事件：

```c
gpio_button_was_pressed(&button);
```

状态适合判断“现在是否按住”，事件适合菜单切换、LED 翻转和参数调整。
