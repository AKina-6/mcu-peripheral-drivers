# Day 02：GPIO 输入与按键软件消抖

## 今日目标

学习 STM32 GPIO 输入模式，实现按键非阻塞消抖，并使用按下事件控制 LED 翻转。

## 实验配置

| 项目 | 内容 |
|---|---|
| MCU | STM32F103C8T6 |
| 按键 | PA0，上拉输入 |
| LED | PB0，推挽输出 |
| 按键有效电平 | 低电平 |
| 消抖时间 | 20 ms |

## 实现步骤

1. PA0 配置为内部上拉输入。
2. PB0 配置为推挽输出。
3. 引入 Day 01 的 `gpio_led` 驱动。
4. 添加 `gpio_button` 驱动。
5. 每 1 ms 更新一次按键状态。
6. 检测按下事件并翻转 LED。
7. 编译、烧录并测试短按和长按。
