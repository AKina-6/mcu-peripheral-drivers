# EXTI 外部中断学习笔记

## 轮询与中断

轮询需要 CPU 持续读取引脚；外部中断在指定边沿出现时通知 CPU。

EXTI 常用于按键、编码器、数据就绪、外部告警和脉冲计数。

## 触发方式

- 上升沿
- 下降沿
- 双边沿

内部上拉、按下接地时，按下对应下降沿。

## ISR 原则

中断中应快速确认来源、记录状态、通知主循环或任务并尽快返回。

复杂处理应延迟到主循环或 RTOS 任务中。

## 按键抖动

EXTI 对每个边沿触发，机械抖动仍会导致多次中断。本实验使用 30 ms 时间窗口过滤重复边沿。

## Tick 溢出

使用无符号减法：

```c
(uint32_t)(now - last)
```

可正确处理 32 位 tick 回绕后的短时间间隔。
