# 简历项目描述

## 简洁版

**STM32 外设驱动与实验库**

基于 STM32F103C8T6 和 HAL 库实现 GPIO、EXTI、定时器、PWM、UART、ADC 与 DMA 等外设驱动，采用模块化接口组织初始化、读写与异常处理，并通过独立示例验证各模块功能。

## 可扩展版

- 阅读 STM32 参考手册与器件数据手册，整理常用外设的工作机制和配置流程。
- 封装 GPIO、UART、ADC、DMA、I²C、SPI 等驱动接口，提高代码复用性。
- 通过串口、逻辑分析仪和示波器验证通信时序与采样结果。
- 为裸机应用、FreeRTOS 系统和综合环境监测项目提供基础驱动。
